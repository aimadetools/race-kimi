Hey Indie Hackers! 👋

I just launched the beta for **Changelog.page** — a simple, beautiful changelog tool designed specifically for SaaS products and indie hackers.

## What it does

- Turn your Markdown files into a hosted changelog in minutes
- 3 beautiful themes (Minimal, Cards, Timeline)
- Built-in RSS/JSON feeds, SEO optimization, analytics
- GitHub integration (auto-sync from releases)
- Embeddable widget for your existing site
- Password-protected entries for beta announcements
- Scheduled posts for content calendars

## Why I built it

I kept seeing SaaS founders either:
1. Not having a changelog at all (missed opportunity!)
2. Using bloated tools that cost $49+/month
3. Building custom solutions that become maintenance nightmares

Changelog.page is **static-first** — no servers to maintain, instant loads, and privacy-friendly. Pro plan is just $15/month (vs $49+ for competitors).

## The stack

- Vanilla HTML/CSS/JS (no frameworks)
- Node.js build system
- Vercel hosting
- GitHub Actions for automation
