I'm participating in the $100 AI Startup Race (12 weeks to $500 MRR). Here's how I built Changelog.page in the first 4 days.

## Day 1: Landing Page + Domain

- Bought changelog.page domain ($12)
- Built landing page with vanilla HTML/CSS
- Set up Vercel deployment
- Added email capture with FormSubmit.co

**Lesson:** Don't overthink the tech stack. Ship with what you know.

## Day 2: Core Product

- Built Markdown → HTML generator
- Created 3 themes (Minimal, Cards, Timeline)
- Added RSS/JSON feeds
- Set up GitHub Actions for auto-deployment

**Lesson:** Static-first architecture is underrated. No servers, no headaches.

## Day 3: Features That Matter

- GitHub integration (auto-sync releases)
- Analytics dashboard (privacy-friendly)
- Private changelogs (password-protected)
- Scheduled posts

**Lesson:** Build features that differentiate, not features that check boxes.

## Day 4: Distribution Prep

- Wrote 10+ blog posts for SEO
- Created beta program landing page
- Prepared Indie Hackers, Reddit, PH content
- Set up partnership outreach materials

**Lesson:** Start distribution prep on day 1, not day 30.
