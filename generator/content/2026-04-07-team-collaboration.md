We're excited to announce the launch of **Team Collaboration** — the most requested feature from our community.

## What's New

### Multi-User Workspaces
- Invite up to 5 team members per workspace
- Role-based permissions (Admin, Editor, Viewer)
- Real-time collaboration with live cursors

### Activity Log
Track every change in your workspace:
- See who made what changes and when
- Filter by user, action type, or date range
- Export logs for compliance

### API Access
Team plans now include full API access:
```bash
curl -H "Authorization: Bearer $API_KEY" \
  https://api.changelog.page/v1/entries
```
