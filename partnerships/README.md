# Partnership Program for Changelog.page

Templates and resources for building partnerships with micro-SaaS founders, affiliates, and integration partners.

---

## Overview

Partnerships are a key growth channel for Changelog.page. This directory contains all the templates and agreements needed to establish and manage partnerships.

### Partnership Types

| Type | Target | Benefit | Status |
|------|--------|---------|--------|
| **Integration Partners** | SaaS tools with changelog needs | Co-marketing, cross-promotion | Ready |
| **Affiliate Partners** | Content creators, influencers | 30% recurring commission | Ready |
| **Beta Partners** | Early adopters | Free Pro for testimonials | Ready |
| **Content Partners** | Bloggers, podcasters | Guest posts, backlinks | Ready |

## Quick Start

### For Integration Partners
1. Identify SaaS tools in adjacent spaces (analytics, auth, hosting)
2. Send personalized pitch using Template A
3. Offer co-marketing opportunity
4. Track in partnership spreadsheet

### For Affiliates
1. Share affiliate program page link
2. Review application (manual approval)
3. Send welcome kit with assets
4. Pay commissions monthly via PayPal/Stripe

### For Beta Partners
1. Identify active indie hackers/SaaS founders
2. Offer 3 months free Pro in exchange for:
   - Honest testimonial (if positive)
   - Feedback call (30 minutes)
   - Potential case study
