## Full Description

```
Changelog.page helps SaaS companies create beautiful changelogs 
from simple Markdown files. No complex setup, no expensive subscriptions.

Perfect for indie hackers and small SaaS teams who want professional 
changelogs without the enterprise price tag.

FEATURES:
✨ 3 Beautiful themes (Minimal, Cards, Timeline)
🚀 GitHub Actions integration for auto-deployment
📊 Built-in privacy-friendly analytics
🔒 Password-protected private entries
📅 Scheduled posts for content planning
🌍 Custom domain support
📱 Fully responsive design
🔔 Embeddable widget with notifications
🎨 RSS & JSON feeds
🔌 Slack/Discord webhook notifications

HOW IT WORKS:
1. Write changelog entries in Markdown
2. Commit to your GitHub repository
3. Auto-deploy via GitHub Actions
4. Get a beautiful, fast static changelog

Pricing: Free for open source, $15/mo for Pro

Built by indie hackers, for indie hackers.
```
