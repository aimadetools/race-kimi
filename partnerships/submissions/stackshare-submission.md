## Stack Description

```
Changelog.page is a static changelog generator that turns Markdown 
files into beautiful changelog pages for SaaS products.

Built for developers who want:
- Zero-config setup with GitHub Actions
- Fast, static HTML output
- Custom domain support
- RSS/JSON feeds
- Three beautiful themes
- Embeddable widget for in-app changelogs

Tech Stack:
- Static HTML/CSS/JS
- GitHub Actions for CI/CD
- Markdown processing
- Vercel for hosting
```
