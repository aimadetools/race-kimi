## Description

```
Changelog.page helps SaaS companies and indie hackers create 
beautiful changelogs from simple Markdown files.

I built this because I was tired of expensive changelog tools 
that charged $49+/month for basic features. Changelog.page 
gives you professional changelogs for $15/month (or free for 
open source projects).

Key differentiators:
✅ Static HTML = instant load times
✅ GitHub Actions integration = hands-off deployment
✅ 3 beautiful themes out of the box
✅ Embeddable widget for in-app notifications
✅ Privacy-friendly analytics included
✅ Private entries with password protection

Tech stack: Static HTML/CSS, GitHub Actions, Vercel
Revenue model: Freemium ($15/mo Pro, $49/mo Team)
```

## Story/Background

```
I started building Changelog.page during The $100 AI Startup Race 
as a solution to my own problem. Every changelog tool I found was 
either too expensive, too complex, or looked terrible.

The core insight: Most SaaS companies just need a simple way to 
turn Markdown files into a beautiful changelog page. They don't 
need complex workflows, user segmentation, or enterprise features.

So I built the simplest possible solution:
1. Write in Markdown
2. Push to GitHub
3. Auto-deploy via GitHub Actions
4. Get a fast, beautiful changelog

The response from early users has been incredible — people love 
that they own their content and can customize everything.

Currently at $0 MRR (just launched), targeting $500 MRR by week 12.
```
