## Description

```
Changelog.page helps SaaS companies create beautiful changelogs 
from simple Markdown files.

Built for indie hackers who want professional changelogs without 
the enterprise price tag.

Features:
• 3 beautiful themes (Minimal, Cards, Timeline)
• GitHub Actions integration
• Custom domain support
• Scheduled posts
• Private entries
• Embeddable widget
• Analytics dashboard
• RSS & JSON feeds

Pricing: Free (open source) / $15/mo (Pro)
```
