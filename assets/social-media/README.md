# Social Media Assets

This directory contains profile images and banners for Changelog.page social media accounts.

## Required Assets

### Twitter/X (@changelogpage)

**Profile Image** - `twitter-profile.png`
- Size: 400x400px
- Format: PNG
- Content: Changelog.page logo/icon
- Background: Dark (#0f172a) or transparent

**Header Banner** - `twitter-header.png`
- Size: 1500x500px
- Format: PNG
- Content:
  - Dark background (#0f172a)
  - Logo on left
  - Tagline: "Beautiful changelogs for SaaS"
  - Website: https://changelog.page
  - Optional: "Launching on Product Hunt soon 🚀"

### LinkedIn (linkedin.com/company/changelogpage)

**Company Logo** - `linkedin-logo.png`
- Size: 300x300px minimum
- Format: PNG
- Content: Changelog.page logo/icon
- Background: White or transparent

**Cover Image** - `linkedin-cover.png`
- Size: 1128x191px
- Format: PNG
- Content:
  - Dark background (#0f172a)
  - Logo on left
  - Tagline: "Beautiful changelogs for SaaS"
  - Subtext: "Turn Markdown into stunning product update pages"
  - CTA: "Get started →"

## Design Guidelines

### Colors
- Primary: #6366f1 (indigo-500)
- Primary Dark: #4f46e5 (indigo-600)
- Background: #0f172a (slate-900)
- Text: #f8fafc (slate-50)
- Text Secondary: #94a3b8 (slate-400)

### Fonts
- Headings: Inter or system sans-serif
- Body: Inter or system sans-serif

### Logo
The logo can be:
1. Text "Changelog.page" in white
2. Icon: 📝 (page with lines) or similar
3. Combination of both

## Creating the Assets

### Option 1: Canva (Recommended)
1. Go to https://www.canva.com
2. Search for "Twitter header" or "LinkedIn banner" templates
3. Customize with brand colors
4. Download as PNG

### Option 2: Figma
1. Create new file
2. Use frames sized correctly
3. Export as PNG @2x

### Option 3: Hire Designer
Platforms: 99designs, Fiverr, Upwork
Budget: $50-100 for all assets

## Upload Instructions

### Twitter/X
1. Go to https://twitter.com/settings/profile
2. Click "Edit profile"
3. Upload profile photo and header
4. Save changes

### LinkedIn
1. Go to https://linkedin.com/company/changelogpage
2. Click "Edit page"
3. Upload logo and cover image
4. Save changes
