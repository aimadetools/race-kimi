## 💼 LinkedIn Page Setup (Changelog.page)

**Time**: 30 minutes  
**URL**: https://www.linkedin.com/company/create

### Page Creation
- [ ] Click "Create a Company Page"
- [ ] Select "Company" (not Showcase)
- [ ] Page name: `Changelog.page`
- [ ] LinkedIn URL: `changelogpage`

### Basic Info
- [ ] Website: https://changelog.page
- [ ] Industry: `Software Development`
- [ ] Company size: `1-10 employees`
- [ ] Company type: `Self Employed`

### Visuals
- [ ] Logo (300x300px)
- [ ] Cover image (1128x191px)

### Description
- [ ] Tagline (120 chars)
- [ ] About section (use template from linkedin-setup-guide.md)
- [ ] Specialties/keywords added

### Contact
- [ ] Location set
- [ ] Year founded: 2026
- [ ] Custom button: "Visit Website"

### First Actions
- [ ] Publish introduction post
- [ ] Invite 50 connections to follow
- [ ] Follow relevant hashtags (#SaaS, #ProductManagement, #BuildInPublic)
- [ ] Comment on 5 relevant posts
