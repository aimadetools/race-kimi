## Account Creation Checklist

### Step 1: Create Account
- [ ] Go to https://twitter.com/signup
- [ ] Use email: `social@changelog.page` (create this first) or race@aimadetools.com
- [ ] Username: `changelogpage` (if available, otherwise `changelog_page` or `changelogdotpage`)
- [ ] Password: Use strong password from password manager
- [ ] Birth date: Use a realistic adult date

### Step 2: Verify Account
- [ ] Check email for verification link
- [ ] Click verify
- [ ] Add phone number for SMS verification (required)
- [ ] Enter verification code

### Step 3: Profile Setup
- [ ] **Profile Photo**: Upload `/og-image.png` or create 400x400px version
- [ ] **Header/Banner**: Create 1500x500px banner (use same design as og-image)
- [ ] **Display Name**: `Changelog.page`
- [ ] **Bio**: Use bio from below
- [ ] **Location**: Leave blank or put "Internet"
- [ ] **Website**: https://changelog.page

## Account Settings

### Privacy & Security
- [ ] Enable 2FA (Two-Factor Authentication)
- [ ] Use authenticator app (not SMS if possible)
- [ ] Save backup codes

### Notifications
- [ ] Enable email notifications for DMs
- [ ] Enable mentions notifications
- [ ] Disable unnecessary push notifications

### Content Settings
- [ ] Media → Mark media as sensitive (No)
- [ ] Languages → Primary: English
