Installation

Add the widget to your site with a single line of code.

HTML 
<!-- Add this before closing </body> tag -->
<script 
  src="https://changelog.page/widget/changelog-widget.js"
  data-feed="https://your-site.com/changelog/feed.json"
  data-position="bottom-right"
  data-theme="dark"
  data-limit="5"
  data-changelog-url="/changelog"
  data-title="What's New">
</script>

JSON Feed Format

The widget expects a JSON Feed v1.1 compatible format.

Example feed.json 
{
  "version": "https://jsonfeed.org/version/1.1",
  "title": "My Product Changelog",
  "home_page_url": "https://example.com",
  "feed_url": "https://example.com/changelog/feed.json",
  "items": [
    {
      "id": "2024-01-15-new-feature",
      "title": "🎉 Introducing Dark Mode",
      "summary": "You asked, we delivered. Switch between light and dark themes.",
      "date_published": "2024-01-15T10:00:00Z",
      "url": "https://example.com/changelog/2024-01-15-new-feature",
      "category": "feature"
    },
    {
      "id": "2024-01-10-bug-fix",
      "title": "Fixed login redirect issue",
      "date_published": "2024-01-10T14:30:00Z",
      "category": "fix"
    }
  ]
}

Category Types

feature New features
improvement Enhancements
fix Bug fixes
security Security updates
announcement Announcements

Item Fields

id — Unique identifier (required)
title — Entry title (required)
summary — Short description
date_published — ISO 8601 date
url — Link to full entry
category — Entry type

Frequently Asked Questions

Do I need to use Changelog.page to use this widget?

No. The widget works with any JSON Feed v1.1 compatible endpoint. If you already have a changelog system that generates a JSON feed, you can use the widget with it.

How does the unread badge work?

The widget stores read entry IDs in the user's browser localStorage. When new entries appear in the feed that haven't been seen, the badge count increases. No server-side tracking required.

Can I customize the colors?

Currently we offer dark and light themes. Full CSS customization is on our roadmap. For now, you can self-host the widget file and modify the CSS variables to match your brand.

Is the widget accessible?

Yes. The widget includes ARIA labels, keyboard navigation (Escape to close), focus management, and respects reduced motion preferences.

Will it slow down my site?

No. The script is ~15KB minified and loads asynchronously. It doesn't block page rendering. The JSON feed is only fetched when the user opens the widget or every 5 minutes for badge updates.
