## Writing Tips

✅ **DO:**
- Lead with user benefits, not features
- Use specific numbers and metrics when possible
- Include visuals (screenshots, GIFs, videos)
- Write in second person ("you can now...")
- Keep paragraphs short (2-3 sentences max)

❌ **DON'T:**
- Use technical jargon without explanation
- List every minor bug fix
- Write long paragraphs
- Forget to include a CTA at the end
