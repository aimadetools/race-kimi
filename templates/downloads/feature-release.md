## Writing Tips for Feature Releases

✅ **DO:**
- Start with the "why" before the "what"
- Use storytelling to create emotional connection
- Include multiple use cases for different personas
- Add clear CTAs throughout (not just at the end)
- Feature customer quotes or testimonials
- Include pricing/availability information upfront

❌ **DON'T:**
- Focus only on technical specs
- Forget to explain who this is for
- Skip the "how to get started" section
- Use only stock photos (show the real product)
- Make users scroll to find pricing info
